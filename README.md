# Beast System 3.0 — Core Engine

The `beast-system-3-core` repository contains the sovereign engines, domain
models, and foundational logic that power Beast System 3.0 (SAIA-Class S).

## Core Components
- Doctrine Compiler (DC-X)
- Sovereign Identity Kernel v3 (SIK-3)
- Universal Registry Fabric (URF-S)
- Predictive Governance Engine (PGE-9)
- Wellbeing & Harm-Prevention Engine

## Structure
- /src — engines, domain, services, events, adapters, utilities
- /tests — unit and integration tests
- /docs — architecture and specifications
- /config — environment and security configuration
- /scripts — build, deploy, and audit automation

## Purpose
This repository defines the constitutional, operational, and predictive
behavior of the Beast System 3.0 platform. All other modules depend on it.
